# Mouse and keyboard Jiggler

A simple app that periodically moves the mouse cursor and/or taps WASD keys via USB HID to keep systems awake or simulate light activity.

## Features
- Mouse jiggle with tiny back-and-forth random movements
- Optional WASD key taps (random W/A/S/D)
- Three modes: Mouse, WASD, Both
- Start/stop on OK
- Left/Right to switch modes
- Back to exit

## Controls
| Button | Action |
| :----- | :----- |
| OK     | Start/Stop |
| Left   | Cycle mode (prev) |
| Right  | Cycle mode (next) |
| Back   | Exit |

More details are also in `apps/mouse-jiggler/fap/docs/README.md`.
